# Fast Food E-Commerce Platform - Project Plan

## Tech Stack
- **Frontend:** React + Vite (User Website & Admin Dashboard)
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Authentication:** JWT (Email/Password) + OAuth (Google/Others)
- **Payment Gateway:** Paystack
- **Image Hosting:** Cloudinary

## Project Structure
```
fastfood-ecommerce/
├── user-frontend/          # Customer-facing website
├── backend/                # API server (serves both user & admin)
├── admin-dashboard/        # Admin panel
└── PROJECT_PLAN.md
```
